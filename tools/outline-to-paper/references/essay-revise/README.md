# Essay Revise Prompts

These prompts are based on pages 172-190 of
`724520134-Writing-Skills-With-Readings-Ninth-Edition.pdf`, which presents
four bases for revising essays:

1. Unity
2. Support
3. Coherence
4. Sentence skills

Use each prompt as a focused reviser that evaluates one dimension of a draft
and suggests stronger revisions.

## Files

- `unity-reviewer.md`
- `support-reviewer.md`
- `coherence-reviewer.md`
- `sentence-skills-reviewer.md`
